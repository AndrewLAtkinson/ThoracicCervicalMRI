Ce CD/DVD provient de l'institution suivante :








Ce CD/DVD a été produit avec le logiciel MediFilm (version 3.5.0.64) de Softinfo (www.softinfo.ca).

Sur ce CD/DVD, vous trouverez :
1) Les images radiologiques en format DICOM.  Celles-ci peuvent être importées dans un système tiers par une clinique ou un hôpital.  La qualité des images a été choisie par l'institution mentionnée ci-haut.  Plus de détails sont disponibles dans le fichier INDEX.HTM.

2) Un contenu web, accessible par le fichier INDEX.HTM, permet d'explorer les fichiers exportés.  Vous y trouverez des informations textuelles ainsi qu'un aperçu des images.  Ce contenu est standardisé XHTML, tout navigateur compatible pourra en afficher le contenu.

3) MediFilmCD (version 3.0.4.0) de Softinfo (www.softinfo.ca).  
- Il s'agit d'un visualiseur DICOM complet qui vous permettra de visualiser et manipuler les images exportées sur ce CD/DVD.  
- Si l'institution ci-haut mentionnée a choisi d'exporter les images de qualité diagnostique, MediFilmCD pourra être utilisé par un médecin pour diagnostiquer.
- MediFilmCD fonctionne sur les systèmes d'exploitation Windows.  Il est optimisé sur Windows XP.

This CD/DVD is provided to you by :








This CD/DVD has been created with MediFilm (version 3.5.0.64) from Softinfo (www.softinfo.ca).

On this CD/DVD, you will find :
1) Radiological images in DICOM format.  Those images can be imported by an hospital or medical clinic into their own PACS system.  The quality of the images has been chosen by the institution mentioned above.  More details are available in INDEX.HTM.

2) A web content, accessible by the file INDEX.HTM, will give you the ability to explore the exported files. You will find there textual information and an overview of images. Such content is standardized XHTML, all compatible browsers should display it correctly.

3) MediFilmCD (version 3.0.4.0) of Softinfo (www.softinfo.ca).
- This is a DICOM viewer that allows you to view and manipulate the exported images on this CD / DVD.
- If the institution mentioned above has decided to export the images in diagnostic quality, MediFilmCD may be used by a doctor to diagnose.
- MediFilmCD runs on Windows operating systems. It is optimized for Windows XP.
